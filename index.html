Study and Implementation of Ontologies through Methods of Automatic Group Labeling

Isaac Silva Santos Ramos (UFPI), Vinicius Ponte Machado (UFPI)

Keywords: Ontology; Labeling; Clustering;

Clustering is an unsupervised machine learning technique that partitions a dataset into finite subsets of elements. However, determining the characteristics influencing the clustering algorithm can be challenging. In light of this, it is necessary to apply labeling algorithms that provide interpretations of the subsets, known as labels.

These labels contain information about the utilized data and serve as the basis for ontology formation. Ontologies represent refined knowledge acquired through the interpretation of these labels. The objective of this project is to return ontologies to the user, formally describing the relationships between elements in a specific context.
